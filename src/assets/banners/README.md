# Banner Assets

This folder contains banner images for the Phố Phường home page.

## Recommended Sizes

- **Large Banner**: 2100x900px (21:9 aspect ratio)
- **Promo Cards**: 800x1000px (4:5 aspect ratio)
- **Video Mock**: 1920x1080px (16:9 aspect ratio)

## Current Placeholders

The app currently uses CSS gradients as placeholders. Replace with actual images:

- `hang-bac-banner.jpg` - Jewelry/Gold banner
- `hang-sac-promo.jpg` - Beauty/Cosmetics promo
- `hang-dien-promo.jpg` - Electronics promo
- `pho-thoi-trang-promo.jpg` - Fashion promo

## Usage

Import in components:

```tsx
import hangBacBanner from "@/assets/banners/hang-bac-banner.jpg";
```
